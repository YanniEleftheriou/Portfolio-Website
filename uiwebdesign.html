CrowdLink
CrowdLink is a networking application designed to help users connect with others at concerts, events, and crowded spaces through shared interests and live interactions. I designed the full product experience in Figma, including user flows, UI layouts, and interaction patterns. After finalizing the visual language, I built a working prototype in code that simulates real-time discovery, messaging, and event-based matching. CrowdLink explores how digital tools can enhance in-person social environments while maintaining safety and control for the user. [Click here to access the project →]
Portfolio Interface
Before building my portfolio website in code, I created a fully realized digital interface in Figma to map out the user flow, visual hierarchy, and micro-interactions. I designed each page layout across multiple breakpoints, incorporating transitions, hover states, and tap interactions to create a functional prototype that mirrors the final coded experience. This process allowed me to refine the structure, pacing, and usability of the site before development.
EcoGauge
EcoGauge is a digital interface concept I designed in Figma for my Creating User Interfaces course. The project reimagines a real-world sustainability tracking program used on my campus, focusing on clarity, accessibility, and visual communication. I redesigned the user flow and dashboard to make environmental metrics more intuitive, with a strong emphasis on color, hierarchy, and data visualization. The prototype demonstrates how a more streamlined UI can make environmental impact tracking easier and more engaging for users.
Signal Lost
Signal Lost is an original project created for my Reality TV course, where the assignment involved developing a fictional reality show, producing a trailer, and crafting social media identities for each cast member. To expand the technical challenge, I designed and built a full website for the show that includes character pages, story summaries, and UI elements that replicate a social media experience. I also developed functioning Instagram-style pages in code, which required carefully recreating the platform’s interface and behavior patterns. This project allowed me to blend storytelling, interface design, and front-end development into a cohesive digital world. [Click here to access the full project →]
Zenith Wellness
Zenith Wellness is a prototype website I designed for a course midterm, representing a hypothetical Pilates and wellness studio. I created a clean, calming interface that highlights core offerings, session details, and brand identity. The design emphasized simplicity, accessibility, and a welcoming tone, using soft colors, balanced spacing, and structured layouts. [Click here to view Zenith Wellness →]
